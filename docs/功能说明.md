# 超市多品类商品协同需求预测系统

## 系统概述
基于LSTM神经网络的超市多品类商品协同需求预测系统，融合商品互补性/替代性协同特征，配合ARIMA基线对比验证，并自动生成库存决策建议。

## 核心功能

### 1. 商品协同分析（互补性 + 替代性）
- 基于销量相关系数矩阵识别商品间关系
- **互补商品**：正相关（如鸡蛋↔牛奶，经常一起购买）
- **替代商品**：负相关（如白菜↔娃娃菜，此消彼长）
- 将互补/替代商品的聚合销量作为LSTM输入特征

### 2. LSTM神经网络预测
- 双层LSTM + SKU嵌入层
- 输入特征：销量、价格、库存、互补销量、替代销量、时间特征（共8维）
- 预测未来7天销量

### 3. ARIMA基线对比
- 使用auto_arima自动选参
- 体现LSTM在捕捉非线性关系上的优势

### 4. 库存决策应用
- 安全库存计算
- 补货建议生成
- 缺货风险评估
- 可视化决策看板
